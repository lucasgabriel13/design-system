# Design System

O projeto Design System foi desenvolvido para simular um design system existente em várias organizações para oferecer componentes que serão utilizado em seus projetos.

Baixe utilizando o NPM:

- [Componentes](https://www.npmjs.com/package/@lg-design-system/react)

- [Tokens](https://www.npmjs.com/package/@lg-design-system/tokens)

Fiquem a vontade para inserir novos componentes! Basta abrir um PR nesse [repositório](https://github.com/lucasgabriel13/design-system).
